# 🍬 Candy Blast - Quick Start Guide

## Play Now (3 Simple Steps)

### 1. Open the Game
Double-click `index.html` or drag it into your web browser.

### 2. Click PLAY
The game will start immediately!

### 3. Match Candies
- **Drag** a candy to swap with an adjacent one
- **Match 3+** of the same color
- Complete objectives before moves run out

---

## 🎮 Game Controls

### Desktop
- **Mouse**: Click and drag candies to swap

### Mobile
- **Touch**: Tap and drag candies to swap

## 🎯 First Time Playing?

### Level 1 Goal
- **Objective**: Score 3,000 points
- **Moves**: 30
- **Stars**: Earn up to 3 stars based on score

### How to Score
- Match 3 candies = 60 points
- Match 4 candies = 120 points + Special Candy
- Match 5+ candies = 200 points + Bomb Candy
- Combos multiply your score!

### Tips
1. Look for matches that create cascades
2. Match candies at the bottom of the board
3. Create special candies when possible
4. Watch your remaining moves

## 🐛 Troubleshooting

### Game won't load?
- Make sure JavaScript is enabled in your browser
- Try a different browser
- Check browser console for errors (F12)

### Sounds not playing?
- Sound files are optional placeholders
- Add your own .mp3 files to `assets/sounds/` folder
- See `assets/sounds/README.txt` for details

### Touch controls not working?
- Make sure you're on a touch-enabled device
- Try refreshing the page
- Check if your browser supports touch events
