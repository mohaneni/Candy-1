CANDY BLAST - FONTS
==================

This game uses Google Fonts CDN for typography.

Fonts used:
- Nunito (weights: 800, 900) - Main UI font
- Fredoka One (weight: 400) - Display/logo font

These fonts are loaded automatically via CDN in index.html.
No downloads or installations required!

The fonts are included via this link in the HTML:
https://fonts.googleapis.com/css2?family=Nunito:wght@800;900&family=Fredoka+One&display=swap

If you want to use custom fonts instead:
1. Download font files (.woff2 format recommended)
2. Place them in this folder
3. Add @font-face rules in css/main.css
4. Remove the Google Fonts link from index.html
